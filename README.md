# DarthBot

Bot en desarrollo

# instalacion

```
git clone https://github.com/WilzTeN/DarthBot.git
cd DarthBot
npm i
```
## Crea un archivo .env con lo siguiente

```
token= your token bot
```
## config JSON
### configura los siguientes parametros

```
clientID: clientId
guildId: guildId
```

# Encendido
## Ejecuta lo siguiente en la consola

```
node slashcommands.js
node start
```
